# PSAM Data

This directory contains sample and test data for PSAM analysis.

## Structure

- `sample/` - De-identified sample data for testing
- `schemas/` - Data schema definitions  
- `test/` - Test datasets for validation

## Data Privacy

⚠️ **Important**: Only de-identified, simulated, or publicly available data should be stored in this repository.

All data in this repository should be:
- De-identified (no personal information)
- Simulated/synthetic data for testing
- Aggregated statistics only
- Publicly available datasets

## Current Data Location

Your actual PSAM data is currently loaded via the MCP server and should NOT be stored in this GitHub repository.
